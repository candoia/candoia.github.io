Candoia

Adoptability and Customizability Study

by Nitin M Tiwari, Ganesha Upadhyaya, Hoan Anh Nguyen, and Hridesh Rajan

Introduction

In this work, we used the Candoia to answer our research questions. Candoia enables developing, customizing and adopting mining software repository(MSR) tools as application. In this document we provide all Java source code, Candoia application code used to generate the data for this study. We also provide the output of running these java programs, which is a chart used for this study. These results were further processed to present them in the paper in a form easier to digest. We provide these programs and data as an archival supplement to the paper, to allow researchers to understand our methodology and, if they wish, use the data to investigate their own research questions.
